Global Fishing Watch Data Standard Operating Procedure

1.2 Available Data

The GFW AIS dataset includes three main types of information about vessels: attributes (such as length, tonnage, flag, and gear type), positions (in WGS84), and activities (like encounters, loitering, port visits, and fishing). Data are available from 2012 - present. Most of the tables with vessel information are related through the source specific vessel id (ssvid).
